Submission Register Method

Built around a practical Submission Register method

Submissions PDF works best when used with a Submission Register, a numbered list of the documents you plan to submit.

That register can include:

  • mandatory response documents
  • supporting attachments
  • extra documents such as an Executive Summary or Cover Letter

Using a numbered Submission Register helps create a cleaner and more controlled submission process because it can:

  • set file order clearly using numeric prefixes
  • keep attachment references consistent across the tender response
  • make final checks easier by letting you compare your ready-to-submit files against the register

Importantly, many teams do not rename all working files to their final numbered names during drafting. That is often helpful, because the same evidence files may be reused across multiple tenders. Submissions PDF supports that approach by helping you apply the final naming structure later, when you are preparing the submission set.

How It Works

Typical workflow for structured submission preparation

  1. Read the tender documents
    Identify the documents to be submitted, including mandatory response files and any supporting attachments.
  2. Build a submission register
    Create a register with numeric prefixes to produce the sort order you want. Use it to track progress for each file, such as Not Started, In Progress, and Completed.
  3. Draft the submission using the register
    Cite attachments by their register file name and prefix, and keep the register updated as you go. There is no need to rename the actual files yet, which is useful when reusing attachments across tenders.
  4. Add your files to Submissions PDF
    Load or drop your Word and PDF files into the app.
  5. Convert Word files to PDF
    Convert Word documents before finalising the submission.
  6. Load your submission register
    Import your submission register into the app.
  7. Scan and review proposed renames
    Match files to register items and review the suggested file names before any changes are made.
  8. Apply renames
    Finalise the file names so your submission is ordered, consistent, and easy to check. The app automatically renames the approved files for you, ready for submission.
  9. Check your submission
    Your files ready for submission should match those in the submission register, making final checks easy.
  10. Combine PDFs if required
    Combine files into a single PDF if the submission requires one consolidated document.
